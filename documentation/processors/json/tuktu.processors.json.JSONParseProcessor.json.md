### tuktu.processors.json.JSONParseProcessor
Parses a field's string content to JSON.

  * **id** *(type: string)* `[Required]`

  * **result** *(type: string)* `[Required]`

  * **config** *(type: object)* `[Required]`

    * **field** *(type: string)* `[Required]`
    - The field containing the value to be parsed to JSON.
